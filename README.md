# PHP Employee & Registration System
## 📌 Project Description
This is a simple User Management System built using PHP and MySQL.  
It allows users to register, login, and manage employee data.


## 🚀 Features
- 🔐 User Registration (with password hashing)
- 🔑 Login Authentication
- 🏠 Dashboard (Home page)
- ✏️ Edit Employee Details
- ❌ Delete Employee
- 🚪 Logout System

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML/CSS

## ⚙️ Setup Instructions
1. Clone the repository
2. Import database in MySQL
3. Update `db.php` with your credentials
4. Run project on XAMPP / localhost
